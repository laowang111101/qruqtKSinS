# 健身房管理系统简介 java773

## 项目概述

本项目是一款基于前后端分离架构的健身房管理系统。该系统旨在为健身房提供全方位的管理解决方案，支持管理员与会员角色的功能划分，实现高效的业务流程和数据处理。

## 技术栈

- 后端：Spring Boot
- 前端：Vue
- 数据库：MySQL
